# InfoWindow
信息窗口组件，可参考官网对于一些[属性](http://lbsyun.baidu.com/cms/jsapi/reference/jsapi_reference.html#a3b7)
可配置属性：
- position
- width
- height
- maxWidth
- offset
- title
- enableAutoPan
- enableCloseOnClick
- enableMessage
- message
- events

事件options：
- close
- open
- maximize
- restore
- clickclose
